# Análise de Campanhas com JavaScript

Aplicação de análise e visualização de dados desenvolvida para demonstrar domínio em lógica, organização e apresentação de resultados.

## Objetivo
Calcular KPIs de campanhas de marketing e mostrar desempenho por canal diretamente no navegador.

## Stack
- HTML
- CSS
- JavaScript

## Como executar
Abra o arquivo `app.html` no navegador.

## O que o projeto mostra
- Total de cliques
- Total de conversões
- Taxa geral de conversão
- Melhor canal
- Tabela com desempenho por campanha
